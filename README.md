NIO Extras
==========

NIO Extras contains extensions to Java NIO API.


Download
--------

Add a dependency to NIO Extras:

    <dependency>
      <groupId>org.jvirtanen.nio</groupId>
      <artifactId>nio-extras</artifactId>
      <version><!-- latest version --></version>
    </dependency>
